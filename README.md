# 🔄 Multi-Purpose Converter App

![Android](https://img.shields.io/badge/Platform-Android-brightgreen.svg) ![Java](https://img.shields.io/badge/Java-v1.8-blue.svg) ![License](https://img.shields.io/badge/License-MIT-green.svg)

The **Converter App** is a versatile Android application that allows users to convert between various units with ease. Designed for speed and accuracy, this app is perfect for students, professionals, and everyday use.

## 🚀 Features

- 📏 **Unit Conversions**:  
  Convert between different units of measurement, including:  
  - Length: Meter ↔️ Kilometer ↔️ Mile ↔️ Inch ↔️ Foot  
  - Weight: Kilogram ↔️ Gram ↔️ Pound ↔️ Ounce  
  - Temperature: Celsius ↔️ Fahrenheit ↔️ Kelvin  
  - Volume: Liter ↔️ Milliliter ↔️ Gallon ↔️ Pint  

- 💱 **Currency Converter**:  
  Real-time exchange rates for major currencies like USD, EUR, GBP, and more (requires internet).

- ⌛ **Time Zones**:  
  Convert between different time zones worldwide.

- ⚡ **Energy Units**:  
  Joules ↔️ Calories ↔️ Kilowatt-hours ↔️ Electronvolts.

- 📐 **Geometry Calculator**:  
  Calculate areas, perimeters, and volumes for various shapes.

- 🔔 **History & Favorites**:  
  Save frequently used conversions and access your conversion history.

- 📱 **Responsive Design**:  
  Optimized for a variety of Android devices and screen sizes.

## 📦 Installation

Follow these steps to get started with the Converter App:

1. **Clone the repository**:
   ```bash
   git clone https://github.com/huzaifaAhmad9/Converter-App.git

2. Open the project: Open the project folder in Android Studio.

3. Sync Gradle: Allow Android Studio to sync Gradle dependencies automatically.

4. Build the project: Navigate to Build > Make Project to ensure no build errors exist.

5. Run the app: Connect an Android device or emulator and click on Run.
